## 📋 Промпт для ИИ (Gemini 3)

### О проекте
Telegram бот для сбора заявок на бесплатный 3-дневный курс от психолога. Бот написан на Python с библиотекой aiogram 3.

### Как работает
1. Пользователь пишет `/start`
2. Бот приветствует по имени и показывает кнопку "Тест 📝"
3. Пользователь проходит опрос из 6 вопросов (выбирает варианты кнопками)
4. После завершения показывается результат и кнопка канала
5. Все ответы сохраняются в SQLite базу данных `responses.db`

### Вопросы
1. Пештар курс харидӣ? (Бале/Не)
2. Агар Бале, барои чӣ харидӣ? (дард доштам, натиҷа мехостам, муаллим писанд омад, нарх хуб буд)
3. Дар он курс чӣ намерасид? (фаҳмондан, амал, мисол, дастгирӣ, натиҷа)
4. Ҳоло бештар кадом мушкил туро азоб медиҳад? (вобастагӣ, хиёнат, беқадрӣ, оромӣ нест, фикри зиёд)
5. Агар ин ҳал шавад, ту бештар чӣ мехоҳӣ? (оромӣ, қувват, муҳаббат, худбоварӣ)
6. Барои ҳал шудани ин мушкил чӣ кадар пул дода метавонӣ? (20$, 50$, 100$, 500$)

### Хостинг
- **Railway** - автоматический деплой с GitHub
- Репозиторий: `https://github.com/sharabulet80-boop/Bot-2-04`
- Токен бота хранится в переменных окружения `BOT_TOKEN` и `ADMIN_IDS`

### Файлы проекта
- `bot.py` - основной код бота (~500 строк)
- `requirements.txt` - зависимости (aiogram, python-dotenv)
- `responses.db` - SQLite база с ответами
- `users.json` - список пользователей
- `Procfile` - для Railway

### Команды
- `/start` - начать
- `/admin` - панель админа (только для админов)

### Админ панель
- Кнопка "📤 Экспорт" - скачать ответы в CSV формате (Excel)

### Автоматизация
Бот сам сохраняет ответы в базу. Админ может скачать CSV файл с красивыми заголовками.
